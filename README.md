# eth-blocks

## Setup

- `git clone git@github.com:aburkut/eth-blocks.git cd eth-blocks`
- `nvm use 14.17`
- `npm install`

### NPM scripts

- `npm run build`
- `npm run start:dev:puller`
- `npm run start:dev:calc`
- `npm run deploy:contract`
- `npm test`
- `npm run lint`
- `npm run lint:fix`
